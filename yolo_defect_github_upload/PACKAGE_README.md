# Submission Package Notes

This package is intended to contain all experiment-related materials except the raw dataset payload.

## Included

- project README and upload notes
- data conversion script
- YOLOv5 code modifications
- training/validation/detection outputs
- curated model weights
- paper figures and captions
- dataset config files and conversion report

## Excluded

- raw GC10-DET images
- raw XML annotations
- YOLO-format image and label payloads
- nested `.git` metadata
- cached files and Python bytecode

## Dataset reference

- Dataset name: `GC10-DET`
- Metadata repository: `datasets/GC10-DET-meta/README.md`
- Training config used in experiments: `datasets/gc10_yolo/data.yaml`
